# PDF to Audiobook Converter

This project allows you to convert any PDF document into an audio file (MP3). While the program runs, it reads the PDF content aloud and also saves the narration as an MP3 file for future use.

## Features
- Converts any PDF into audio format (MP3)
- Reads the content of the PDF aloud in real-time
- Saves the audio to an MP3 file for easy listening later

## How to Use
1. Clone this repository to your local machine.
2. Install the necessary dependencies (see below).
3. Run the script and input the path to your PDF file.
4. The program will read the content aloud and save the audio to an `my_pdf_audio.mp3` file.
